Simple Alert

flavr is designed to be able to run with minimal cofiguration

Code Demo 
new $.flavr('Hello World!');
Confirm Dialog

A confirm dialog with two preset buttons

Code Demo 
new $.flavr({
    content     : 'Are you sure to delete your memories?',
    dialog      : 'confirm',
    onConfirm   : function(){
        alert('Deleted');
    },
    onCancel    : function(){
        alert('Canceled');
    }
});

/*----- OR ----- */

new $.flavr().confirm('Are you sure to delete your memories?', function(){ 
    alert('Deleted');
}, function(){
    alert('Canceled');
});
Prompt Dialog

A better prompt dialog

Code Demo 
new $.flavr({
    content     : 'How old are you?',
    dialog      : 'prompt',
    prompt      : { placeholder: 'Enter something' },
    onConfirm   : function( $container, $prompt ){
        alert( $prompt.val() );
        return false;
    },
    onCancel    : function(){
        alert('Canceled');
    }
});

/*----- OR ----- */

new $.flavr().prompt('Enter something', 'How old are you?', function( $container, $prompt ){
    alert( $prompt.val() );
    return false;
}, function(){
    alert('Canceled');
});
Multiple Dialog Instance

flavr supports unlimited dialog instance/layer that will stack on the user screen

Code Demo 
new $.flavr({
    content  : 'Nuff siad, here is the last one', 
    position : 'top-right'
}, function(){
    new $.flavr({
        content  : 'The stack is unlimited, pretty cool huh',
        position : 'top-left'
    }, function(){
        new $.flavr({
            content  : 'Here is the dialog #3',
            position : 'bottom-right'
        }, function(){
            new $.flavr({
                content  : 'Me dialog #2',
                position : 'bottom-mid'
            }, function(){
                new $.flavr({
                    content :'I am dialog #1',
                    buttons : {
                        exit    : { text: 'Close All', style: 'danger',
                                    action: function(){
                                        this.exit();
                                    }
                        },
                        close   : {}
                    }
                });
            });
        });
    });
});

HTML Content

flavr accepts HTML contents by default. Set the html option to false to prevent flavr accepting any HTML tags.

Code Demo 
var html =  
'   <div class="form-row">' +
'       <input type="text" name="username" ' +
'       placeholder="Username" />' +
'   </div>' +
'   <div class="form-row">' +
'       <input type="password" name="password" ' +
'       placeholder="Password" />' +
'   </div>' +
'   <div class="form-row">' +
'       <input type="checkbox" name="remember" ' +
'       id="check"/>' +
'       <label for="check">Remember me</label>' +
'   </div>';
        
new $.flavr({                
    title       : 'Form',
    iconPath    : 'images/',
    icon        : 'email.png',
    content     : 'HTML form example',
    dialog      : 'form',
    form        : { content: html, method: 'post' },
    onSubmit    : function( $container, $form ){
        alert( $form.serialize() );
        return false;
    }
});
Modeless Dialog

Modeless is the reverse state of modal dialog which doesn't prevent the user to interact to the rest elements of the page. Modeless dialog might not working on IE 8 or any browser that didn't support pointer-events property

Code Demo Close 
var modeless = new $.flavr({
    modal   : false,
    content : 'This is a modeless dialog'
});

/* Closing the dialog */
modeless.close();
Autoclosed Dialog

The following dialog will be auto-closed in 5 seconds

Code Demo 
new $.flavr({
    content     : 'Autoclosing the dialog in 5 seconds',
    autoclose   : true,
    timeout     : 5000  /* Default timeout is 3 seconds */
});
Alternative Closing

flavr provides option to alternatively close the current dialog by clicking the overlay or pressing the keyboard Escape button

Code Demo 
new $.flavr({
    content      : 'Try clicking the overlay or pressing the escape button',
    closeOverlay : true,
    closeEsc     : true
});
Setting up Animation

flavr uses the popular animate.css library created by Daniel Eden to provide some amazing CSS3 animation.

Code Demo 

var select = $(this).parent().siblings('.demo-select');
var entrance = select.find('.animate-entrance :selected').text();
var closing = select.find('.animate-closing :selected').text();
                
new $.flavr({
    animateEntrance : entrance,
    animateClosing  : closing,
    content         : 'Animation example'
});

Dynamic Dialog

Amazingly, flavr allows you to make a fullscreen dialog, resize the dialog to specific height or width, change its content, position and much more, on the fly

Code Demo 
new $.flavr({
    content     : 'Resize your Dialog',
    buttons     : {
        resize  : { text: 'Resize 700 x 300', style: 'info',
                        action: function(){
                            this.resize( 700, 300 );
                            return false;
                        }
        },
        full    : { text: 'Fullscreen', style: 'warning', 
                    action: function(){
                        this.fullscreen();
                        return false;
                    }
        },
        revert  : { text: 'Original Size', style: 'danger',
                    action: function(){
                        this.revert();
                        return false;
                    }
        },
        content : { text: 'Change Content', style: 'info',
                    action: function(){
                        this.content('<p>This is the new content</p><br/><p>New line</p>');
                        return false;
                    }
        },
        close   : {}
    }
});
Throw Callbacks

flavr provides you the ability to throw callbacks at a particular stage and conditions which will greatly help you building an advance dialog

Code Demo Show 
new $.flavr({
    content     : 'flavr callback examples',
    buttons     : {
        shake   : { style: 'primary', 
                    action: function(){
                        this.shake(function(){
                            alert('Shake animation callback');
                        });
                        return false;
                    }
        },
        hide    : { style: 'info', 
                    action: function(){
                        this.hide();
                        return false;
                    }
        },
        close   : { style: 'warning' }
    },
    onLoad      : function(){
        alert('The plugin has been loaded');
    },
    onBuild     : function(){
        alert('We have succesfully build the DOM elements');
    },
    onShow      : function(){
        alert('The dialog has been shown');
    },
    onHide      : function(){
        alert('Now hidding the dialog');
    },
    onClose     : function(){
        alert('We\'re about to close the dialog');
    }
});
